# Node-Blog
A blog application made with Node.js, PUG and Sequelize.

![picture alt](http://i.imgur.com/L2iAtvh.png "Homepage screenshot")

#Features
* Register
* Login
* Logout
* Create a post
* View a specific post
* Leave comment on a post
* View all posts
* View posts of logged in user
